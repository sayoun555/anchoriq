# AnchorIQ — API 응답 가이드라인

> 공통 응답 형식 + 도메인별 핵심 데이터 구조 방향
> **참고용** — 필드는 도메인 모델에서 자연스럽게 도출할 것, 여기에 묶이지 말 것

---

## 공통 응답 형식 (필수)

> 이 3가지 형식만 지킨다. 나머지 data 내부는 자유.

### 성공 응답

```json
{
  "success": true,
  "data": { ... },
  "timestamp": "ISO 8601"
}
```

### 에러 응답

```json
{
  "success": false,
  "error": {
    "code": "에러코드 (SCREAMING_SNAKE_CASE)",
    "message": "유저에게 보여줄 메시지"
  },
  "timestamp": "ISO 8601"
}
```

### 페이지네이션 응답

```json
{
  "success": true,
  "data": {
    "content": [ ... ],
    "totalElements": 0,
    "totalPages": 0,
    "page": 0,
    "size": 20
  },
  "timestamp": "ISO 8601"
}
```

## 에러 코드 목록 (필수)

| 코드 | HTTP 상태 | 의미 |
|------|----------|------|
| `INVALID_CREDENTIALS` | 401 | 이메일/비밀번호 불일치 |
| `TOKEN_EXPIRED` | 401 | JWT 만료 |
| `ACCESS_DENIED` | 403 | 권한 없음 |
| `PLAN_LIMIT_EXCEEDED` | 403 | 플랜 제한 초과 (Free 일 5건) |
| `NOT_FOUND` | 404 | 리소스 없음 |
| `DUPLICATE` | 409 | 중복 (이메일, 관심 선박 등) |
| `PAYMENT_FAILED` | 402 | 결제 실패 |
| `CONCURRENT_MODIFICATION` | 409 | 낙관적 락 충돌 |
| `EXTERNAL_API_ERROR` | 502 | 외부 API 호출 실패 |
| `INTERNAL_ERROR` | 500 | 서버 내부 오류 |

## 도메인별 핵심 데이터 방향

> 정확한 필드를 정의하지 않음. "이런 데이터가 포함되어야 한다" 수준만 가이드.

### 선박 (Vessel)

```
조회 시 포함되어야 할 것:
- 식별: IMO, MMSI, 이름
- 분류: 선박 타입, 국적(flag)
- 상태: SAILING/ANCHORED/MOORED, 리스크 레벨
- 위치: 위경도, 속도, 방향, 갱신 시각 (Redis GEO에서)
- 관계: 소유 회사, 현재 항로 (필요 시 Neo4j에서)
```

### 리스크 (Risk)

```
알림:
- 알림 타입, 리스크 레벨, 대상 선박/초크포인트
- 사유, 추천 액션, AI 신뢰도
- 상태 (PENDING/ACKNOWLEDGED/DISMISSED)

스코어:
- 대상 (선박/항로/초크포인트/항만)
- 종합 점수 0~100
- 항목별 점수 (제재, 기상, 지정학, 항로, 이상)
- 판단 근거 (사람이 읽을 수 있는 텍스트)
```

### AI 질의

```
요청: 자연어 문자열
응답:
- 자연어 답변
- 관련 엔티티 목록 (선박, 항만 등)
- 잔여 질의 횟수 (Free 플랜)
```

### AI 브리핑

```
- 날짜, 요약 텍스트
- 하이라이트 (타입, 제목, 설명, 리스크 레벨)
- 통계 (추적 선박 수, 리스크별 건수, 이상 탐지 건수)
```

### What-if 시뮬레이션

```
요청: 시나리오 텍스트, 기간
응답:
- 영향 분석 (선박 수, 항로, 예상 지연, 추가 비용)
- 대체 항로 (추가 거리, 시간, 비용)
- 추천 사항 (사람이 읽을 수 있는 텍스트 리스트)
```

### 대시보드

```
요약:
- 선박 총 수, 활성 알림 수
- 리스크 분포 (CRITICAL/HIGH/MEDIUM/LOW)
- 초크포인트 6개 현재 상태
- 데이터 소스별 최종 갱신 시각
```

### 결제/구독

```
플랜 목록: 이름, 가격, 통화, 기능 목록, 제한
구독 결과: 구독 ID, 플랜, 상태, 게이트웨이, 금액, 시작/만료일
```

### 지도

```
선박 위치 (경량):
- IMO, 위경도, 선박 타입, 리스크 레벨
- 이름/상세 정보는 클릭 시 별도 API로 조회
```
