## 뷰 구성

| 뷰 | 역할 | 팔란티어 대응 |
|----|------|-------------|
| 지도 뷰 | 선박 실시간 위치 + 리스크 히트맵 | Gotham 지도 |
| 그래프 뷰 | 엔티티 관계 탐색 | Gotham 그래프 |
| 타임라인 뷰 | 이벤트 → AI 판단 → 액션 이력 | Foundry 타임라인 |
| 워룸 대시보드 | 실시간 알림 + 리스크 현황판 | Gotham 워룸 |
| 자연어 질의 바 | AI에게 질문 | Foundry AIP |

## 지도 뷰

```
┌──────────────────────────────────────────────┐
│  [세계 지도]                                  │
│                                              │
│   🔴 리스크 높은 선박     ● 일반 선박          │
│   ⚠️  초크포인트 마커     █ 리스크 히트맵       │
│                                              │
│   [호르무즈] [말라카] [수에즈] [바브엘만데브]    │
│   [대만해협] [파나마]                          │
│                                              │
│   선박 클릭 → 상세 패널 (정보 + 리스크 + 항로) │
└──────────────────────────────────────────────┘
```

- 라이브러리: Mapbox GL JS 또는 Leaflet
- 선박 위치: Redis GEO → WebSocket으로 프론트에 푸시
- 히트맵: 리스크 스코어 기반 색상 (초록 → 노랑 → 빨강)

## 그래프 뷰

```
┌──────────────────────────────────────────────┐
│                                              │
│          [Company A]                         │
│           ↙       ↘                          │
│    [Vessel 1]   [Vessel 2]                   │
│        ↓             ↓                       │
│   [Route: Asia-EU]  [Port: Busan]            │
│        ↓                                     │
│   [Chokepoint: Suez]                         │
│                                              │
│   노드 클릭 → 관계 펼침 (4홉까지)              │
└──────────────────────────────────────────────┘
```

- 라이브러리: D3.js 또는 Cytoscape.js
- 데이터: Neo4j Cypher 쿼리 결과를 그래프로 렌더링
- 인터랙션: 클릭 → 확장, 더블클릭 → 상세, 드래그 → 이동

## 타임라인 뷰

```
┌──────────────────────────────────────────────┐
│ 14:30  ⚡ 이벤트: 태풍 "란" 접근 감지          │
│        └─ 영향 해역: 동중국해                  │
│                                              │
│ 14:31  🤖 AI 판단: 리스크 HIGH                │
│        └─ 근거: 태풍 경로 + 부산항 혼잡도 85%   │
│        └─ 추천: 광양항 우회                    │
│                                              │
│ 14:32  🔔 액션: Slack 알림 발송               │
│        └─ 대상: 운항팀 채널                    │
│        └─ n8n 워크플로우 ID: #1234            │
└──────────────────────────────────────────────┘
```

- 이벤트 → AI 판단 → 액션 실행 전체 흐름 추적
- 데이터: Elasticsearch ai-decisions 인덱스

## 자연어 질의 바

```
┌──────────────────────────────────────────────┐
│ 🔍 "호르무즈 해협 근처 제재 연관 선박?"         │
├──────────────────────────────────────────────┤
│                                              │
│ AI 응답:                                     │
│ 현재 호르무즈 해협 반경 50km 내에 제재 연관      │
│ 선박 2척이 탐지되었습니다.                      │
│                                              │
│ 1. VESSEL-A (IMO: 9811000) — 이란 국적       │
│    └─ 리스크: HIGH, 위치: 26.5°N 56.0°E      │
│ 2. VESSEL-B (IMO: 9722000) — 북한 연관 회사   │
│    └─ 리스크: HIGH, 위치: 26.8°N 56.3°E      │
│                                              │
│ [지도에서 보기] [그래프 탐색] [보고서 생성]       │
└──────────────────────────────────────────────┘
```

- OpenClaw가 자연어 → Cypher 쿼리 변환 → Neo4j 조회 → 자연어 응답 생성
